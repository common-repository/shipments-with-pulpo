PulpoWMS plugin helps you to sync your woocommerce with Pulpo WMS app to manage products, stock and shipping orders.

== Description ==

PulpoWMS plugin helps you to sync your woocommerce with Pulpo WMS platform.
This plugin syncs all information about products, stock management and shipping orders, from your woocommerce with Pulpo WMS app without manual user intervention.

== Configuration ==

Configuration is easy, you only need a credential key to access PulpoWMS platform.

== Frequently Asked Questions ==

= How can I start to use TwenTIC's plugin to synchronise with PulpoWMS in WooCommerce? =
To use this plugin, you need to have a valid credential key to access PulpoWMS platform

= I already have configured credentials, What do I have to do to make it work? =
To make the plugin work, once it is configured, you need clic on "Conf. webhooks" and "Add products", in this order, to sync with pulpo for first time.

= Is the use of the plugin invasive for store customers? =
No, the use of the plugin doesn't interfere with navigation or checkout process.

= Do you have any doubts about the plugin? =
Please share them in our support forum

= What uses does the plugin make of the establishment or end-user's data? =
The plugin shares with Pulpo the details about orders to manage shipment and stock. These details are not shared with third party companies or services through the plugin
